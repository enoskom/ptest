## 🛠️ Kurulum

Projeyi yerel olarak çalıştırmak için şu adımları takip edebilirsiniz:

Projeyi klonlayın:

	git clone https://github.com/enoskom/ptest

Proje dosyasına girin:

	cd ptest

Türkçe çeviriyi incelemek için istediğiniz dosyayı açabilirsiniz.
